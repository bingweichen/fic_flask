# fic_flask
FIC Flask project for Cookiecutter.


## Quick Start
Install Cookiecutter globally:
```bash
$ pip install cookiecutter
```
Generate the boilerplate:
```bash
$ cookiecutter https://github.com/bingweichen/fic_flask.git
```
